Colophon

How This Place Is Built

Jewitch is a small, personal website built with Markdown, stubbornness, and a fondness for the parts of the web that still feel handmade.

Writing

Markdown

Posts and pages begin as Markdown files, which keeps the writing portable, readable, and mine.

Generator

Neato

The site is generated with Neato, Adam Newbold's very early alpha static-site project.

Source

GitHub

The source lives in GitHub, where changes can be tracked, rebuilt, and pushed into the world.

Hosting

Cloudflare Pages

The finished static site is hosted on Cloudflare Pages, with GitHub Actions handling the rebuild.

Editing

TinaCMS

Blog posts can be drafted and edited through TinaCMS, which makes the writing flow feel less like a chore.

Design

Velvet Moon

The look is dark, soft, warm, a little witchy, and built around readable words rather than endless feeds.

Site Philosophy

This site is intentionally personal. It is not optimized for algorithms, growth hacking, or performing professionalism. It is a home for writing, memory, anger, delight, and the kind of internet that still leaves fingerprints.
