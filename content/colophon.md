<p class="page-kicker">Colophon</p>
<h1>How This Place Is Built</h1>
<p class="page-lede">Jewitch is a small, personal website built with Markdown, stubbornness, and a fondness for the parts of the web that still feel handmade.</p>
</section>

<section class="tool-grid" aria-label="Site tools and stack">
<article class="tool-card">
<span><i class="fa-solid fa-file-lines" aria-hidden="true"></i> Writing</span>
<h2>Markdown</h2>
<p>Posts and pages begin as Markdown files, which keeps the writing portable, readable, and mine.</p>
</article>

<article class="tool-card">
<span><i class="fa-solid fa-wand-magic-sparkles" aria-hidden="true"></i> Generator</span>
<h2>Neato</h2>
<p>The site is generated with Neato, Adam Newbold's very early alpha static-site project.</p>
</article>

<article class="tool-card">
<span><i class="fa-brands fa-github" aria-hidden="true"></i> Source</span>
<h2>GitHub</h2>
<p>The source lives in GitHub, where changes can be tracked, rebuilt, and pushed into the world.</p>
</article>

<article class="tool-card">
<span><i class="fa-solid fa-cloud" aria-hidden="true"></i> Hosting</span>
<h2>Cloudflare Pages</h2>
<p>The finished static site is hosted on Cloudflare Pages, with GitHub Actions handling the rebuild.</p>
</article>

<article class="tool-card">
<span><i class="fa-solid fa-pen-to-square" aria-hidden="true"></i> Editing</span>
<h2>TinaCMS</h2>
<p>Blog posts can be drafted and edited through TinaCMS, which makes the writing flow feel less like a chore.</p>
</article>

<article class="tool-card">
<span><i class="fa-solid fa-palette" aria-hidden="true"></i> Design</span>
<h2>Velvet Moon</h2>
<p>The look is dark, soft, warm, a little witchy, and built around readable words rather than endless feeds.</p>
</article>
</section>
